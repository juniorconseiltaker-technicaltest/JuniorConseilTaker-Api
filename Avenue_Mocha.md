## Getting started
- Clone the [repo](https://github.com/juniorconseiltaker-technicaltest/Avenu_Mocha_API)
- cd Avenu_Mocha_API
- Launch the server:
  - docker compose up -d
- Stop the server:
  - docker compose down

- baseURL=http://localhost:3000


### REQUESTS

#### GET

- Get all coffees: 
  - route : /coffees
- Get hot coffees: 
  - route : /coffees/hot
- Get iced coffees: 
  - route : /coffees/iced
- Get coffee shops: 
  - route : /shops
