## Getting started
- Clone the repo
- cd avenue-mocha-server
- Launch the server
  - docker compose up -d
- Stop the server:
  - docker compose down

- baseURL=http://localhost:3000


### REQUESTS

#### GET

- Get all coffees: 
  - route : /coffees
- Get hot coffees: 
  - route : /coffees/hot
- Get iced coffees: 
  - route : /coffees/iced
- Get coffee shops: 
 - route : /shops
